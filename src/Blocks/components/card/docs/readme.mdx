# Card Component

Component used to provide card functionality to your project.

## Dependencies

1. components/image
2. components/heading
3. components/paragraph
4. components/button

## Implementation

Open a terminal and type in this command inside your projects root:

```shell
wp boilerplate use_component --name=card
wp boilerplate use_component --name=image
wp boilerplate use_component --name=heading
wp boilerplate use_component --name=paragraph
wp boilerplate use_component --name=button
```
